# Vehicle Survey

### How to build
In project directory: 
* MAC
```sh
$ ./gradlew clean build
```
* Windows
```sh
$ gradlew.bat clean build
```
Artifact is built into build/libs folder

### How to run
```
java -jar <path-to>/vehicle-survey.jar "<path-to>/datafile"
```

### Input

Choose from menu
```
0 - Menu
1 - Count by day
2 - Count average
3 - Peak volume times
4 - Speed
5 - Distance
6 - Exit
```
